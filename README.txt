Главный файл - vdsinit.sh.
В нем собраны все необходимые команды для базовой настройки VDS и установки nginx, php-fpm и mysql server

Для работы, откройте этот файл и установите переменные в начале файла:

USER, GROUP, SITENAME, DBNAME, DBUSER, DBUSERPASSWORD, UBUNTUCODENAME

После этого сохраните файл и запустите от имени root пользователя на сервере. Он произведет все необходимые установки пакетов, произведет все настройки.
В процессе работы скрипт будет запрашивать необходимые данные: пароль от root пользователя mysql при установке mysql server и запросы типа Y/n.
